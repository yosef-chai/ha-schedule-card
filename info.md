# Schedule Card

A Lovelace custom card for Home Assistant that visualizes `schedule.*` entities
as a 24-hour timeline with **daily / weekly** and **full / compact** views.

Built with Lit 3, TypeScript, and the official `ha-form` UI editor. Ships with
English + Hebrew localization and full RTL support out of the box.

Element name: `ha-schedule-card` — use `type: custom:ha-schedule-card`.

See [README](README.md) for installation and configuration.
